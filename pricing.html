Frequently Asked Questions

Which AI platforms does ContextShift support?
ContextShift works across ChatGPT, Claude, Gemini, Grok, and Perplexity out of the box.
Does it store my conversations?
Screenshots are processed locally. We never store raw conversation data. Summaries are cached on-device only.
Does it work on Windows?
Yes! We provide packaged installers for both macOS (.dmg) and Windows (.exe).
Can I use my own API key?
Yes. Power users can bring their own Gemini or Anthropic API keys to bypass rate limits entirely.
How is this different from copying my chat?
ContextShift extracts the structured state — goal, progress, decisions, next steps. Pasting a raw 40-page chat log into a new AI causes context collapse. We solve that.
